博客框架Hexo简明教程
发布日期:   2019-09-07
阅读次数:  

[1]简介

Hexo是一款静态博客框架，在无需数据库及后端语言的情况下，可通过Markdown格式生成静态页面。

[2]安装

Hexo依赖NodeJS(Javascript运行环境)及GIT(版本控制系统)，安装该两项后才可安装Hexo。
2.1 NodeJs的安装:打开官网，下载安装包，安装后打开cmd命令行，输入node -v,如正确输出版本号即代表安装成功。
2.2 GIT的安装:打开官网，同样下载安装包，安装后打开cmd命令行，输入git --version,如正确输出版本号即代表安装成功。
2.3 Hexo的安装:安装以上软件后，打开cmd命令行，输入npm install hexo-cli -g,等待安装过程，安装后输入hexo -v，如有输出版本号即代表安装成功。

[3]使用

Hexo的使用相对简单，初始化一个仓库，创建博客文章，然后再生成静态文件，通过启动本地服务即可在浏览器端看到博客内容。
