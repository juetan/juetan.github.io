博客框架Hexo简明教程
发布日期:   2019-09-07
阅读次数:  

[1]简介

Hexo是一款静态博客框架，在无需数据库及后端语言的情况下，可通过Markdown格式生成静态页面。

[2]安装

Hexo依赖NodeJS(Javascript运行环境)及GIT(版本控制系统)，安装该两项后才可安装Hexo。

  1. NodeJs的安装: 打开官网，下载并安装，输入node -v,如正确输出版本号即代表安装成功。
  2. GIT的安装: 打开官网，同样下载并安装，输入git --version,如正确输出版本号即代表安装成功。
  3. Hexo的安装: 安装以上软件后，输入npm install hexo-cli -g，再输入hexo -v，如有输出版本号即代表安装成功。

[3]使用

Hexo的使用相对简单，初始化一个仓库，创建博客文章，然后再生成静态文件，通过启动本地服务即可在浏览器端看到博客内容。接下来，以本人博客为例，介绍从无到有的博客生成过程。

  1. 初始化博客仓库
    在你想放置博客仓库的文件夹内，输入以下命令，会在当前文件夹内生成名为jonten的文件夹。

     hexo init jonten  //格式： hexo init <folder> 参数： folder(博客文件夹名字)

    以上命令执行后，打开刚刚生成的文件夹，可以发现文件夹内已经放置好Hexo博客框架，接下来介绍各文件目录及其作用。

     // /joten
 ├── _config.yml  //配置文件 
 ├── package.json //依赖文件
 ├── scaffolds    //模板文件夹
 ├── source       //文章文件夹
 |   ├── _drafts  //草稿
 |   └── _posts   //文章
 └── themes       //主题文件夹

    _config.yml

  2. 创建博客文章

     hexo new hello-jonten
