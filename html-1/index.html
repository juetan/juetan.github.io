简介

  html,即超文本标记语言(Hytype Text Markup Language),一门描述文档内容结构的标记语言。区别于JavaScript，Java，C语言等语言，html并不是一门编程语言，而是一门标记语言。编程语言，具备算法及数据结构，能够执行代码并产生结果，而标记语言既没有算法也没有数据结构，仅仅是对内容标记。所谓标记语言，就是用某个标签对文档中某段内容加以标记，使其包含某种语义。例如，‘绝弹博客’只是一段普通文本，而加上标签<h1>绝弹博客</1h>，便赋予这段文本标题的含义(表示这是一个标题)。而超文本，意味着不仅可以标记文本，也可以标记文本之外的内容。没错，链接、图片、音频、视频都可以标记，只是与文本的标记略有区别。例如，显示一张图片，可以用<img src="./hello.jpg">表示。img是image的缩写，表示img标签里面的内容是一张图片。src是source的缩写，表示资源所在的地址。两者结合起来便是，从当前文档所在的目录下，加载名为hello的jpg图片，并在页面中显示。

文档

  打开浏览器，访问www.google.com，可以看到所呈现的页面。对于初学者，网页、页面、文档，三者可以初略视为等价。其实，文档有许多种类型，常见的有xml、html、xhtml等。但在网页中使用的，均为html类型的文档，其他类型的文档，因为某些原因都终将慢慢淡出历史舞台。对于浏览器来说，加载一份文档后，首先需要知道文档的类型，从而选择合适的引擎解析文档。因而，每一份文档都需要在文档的开头声明本身的文档类型。html有html 4.01和html 5两个流行的版本，两者有所不同(后面会将到),不过后者的光芒已经掩盖前者，终将一统html的天下。

    声明html 4.01，使用<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">
    声明html 5，   使用<!DOCTYPE HTML>

没有对比就没有伤害，html 5的声明如此简洁清晰。不过，怎么看都好像是html 5对html 4.01做出某种不可描述的事情(不掐头只去尾的操作),然后跑出来自立门户的(事实的确是这样，不过后来重新回归)。相较来说，html 5支持的标签更多，语义化的支持也更好，html 4.01已是廉颇老矣(htmo 4.01 心理：当初叫人家小甜甜，现在居然嫌人家过时)，推荐大家使用 html 5的文档声明。

标签

  在声明文档类型为html后，便可以使用html标签对文档内容进行操作。前面提到过标签，标签由左右括号和斜杠包裹标记而成，可分为开始标签和闭合标签，如<p>是开始标签，</p>是闭合标签，有些标签没有只有开始标签没有闭合标签。不同的标签具有不同的语义。例如，<a>标签代表超链接(取anchor首字母，有闭合标签)，<p>标签代表段落(取paragraph首字母，有闭合标签),<img>标签代表图片(取image缩写，没有闭合标签)等。

元素

  元素由标签延伸而来(标签包裹内容形成)，包含更多的信息和结构(如属性、闭合标签)。例如，<time datetime="2019-10-01">国庆节</time>便是一个完整的元素，<time>和</time>分别为开始标签和闭合标签，两者中间包裹的内容成为元素内容(国庆节为元素内容)；datetime="2019-10-01"称之为属性，只出现在开始标签内，datetime是属性的名字，2019-10-01是属性的值(属性值要用双引号括起来，单引号也可以，不过更推荐双引号)，两者用=连接。
  标签可以嵌套，元素也是嵌套的，例如<p>注意:<em>小心地滑</em></p>。层层嵌套下来，文档的结构就像树状一样，既然是树状，总要有树根吧，文档的树根称之为根元素。根元素由<html>``</html>标签及其内容组成，所有文档内所有元素都必须是根元素的子元素，也就是说树枝可以有很多，树根只能有一个。
  根元素下，分别有head和body两个子元素，他们分别代表不同的结构。head元素的内容一般不在页面中显示，主要是声明文档的各种元数据，如字符编码，文章标题，加载样式，加载脚本等；而body元素的内容便是我们在浏览器窗口中看到的内容，不同的元素自带不同的默认样式(h1元素默认字体加粗加大，默认样式根据浏览器的不同而不同)，可以通过编写自定义样式使页面展示更加绚丽。

练习

  简单了解一番后，心中不免有些躁动，迫不及待准备尝试一番，写点什么好呢，来首诗吧。

<!DOCTYPE HTML> <!-- 文档声明 -->
<html>  <!-- 根元素 -->
    <head> <!-- head元素 -->
        <meta charset="USTF-8"> <!-- 字符编码 -->
        <title>古诗</title>   <!-- 文档标题 -->
    </head>
    <body> <!-- body元素 -->
        <h1>鹿柴</h1>
        <p>空山不见人，但闻人语响。</p>
        <p>返景入深林，复照青苔上。</p>
    </body>
</html>
