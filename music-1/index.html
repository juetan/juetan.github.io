环境搭建
发布日期:   2019-10-23
阅读次数:  

1.Node安装

  1. 登录官网，下载最新版本并安装；
  2. 在我的电脑内，执行以下操作：右键->属性->高级->环境变量->系统变量->path 查看是否有node的安装路径，没有的话添加上node的安装路径；
  3. 开始->运行->cmd 输入node -v ，出现类似v10.15.3的字样说明安装成功。

2.vue-cli安装

  1. 打开cmd命令行，输入npm install vue-cli -g，等待安装完成；
  2. 完成后，输入 vue -V，出现类似2.9.6的字样表明安装成功。

2.初始化项目

  1. 在准备放置项目的文件夹里，打开cmd并输入vue init webpack music(music为项目目录名字，可自定义)，按照命令输入相应的内容；
  2. 完成后，依次输入cd music(项目目录)，npm run dev命令，等待完成；
  3. 完成后，安装给出的地址，在浏览器输入localhost:8080，可访问表明初始化成功；
  4. 在安装文件夹里，music目录下包含如下内容：
     //music文件夹
 build   --打包工具及开发配置
 config  --配置目录
 nodemodules --依赖模块
 src --源代码目录
     assets --静态文件
     components --组件
     App.vue --应用级组件
     main.js --入口文件
 static --静态文件
 index.html --首页
 //部分文件于项目关系不大，故此省略
